# Logo Prompt Guide - Indien Rejsedagbog

## TYPE
Emblem logo / Badge logo med ikon og tekst

## FORM
Cirkulær baggrund med centreret ikon (tuktuk)

## MOTIV
Indisk tuktuk (auto-rickshaw) set fra siden

## FARVEPALET

| Farve | HEX | RGB | Brug |
|-------|-----|-----|------|
| Saffron Orange | #FF9933 | 255, 153, 51 | Primær / baggrund |
| India Green | #138808 | 19, 136, 8 | Sekundær / tag |
| Navy Blue | #000080 | 0, 0, 128 | Accent / hjul |
| Hvid | #FFFFFF | 255, 255, 255 | Tuktuk karosseri |
| Himmelblå | #87CEEB | 135, 206, 235 | Vindue |
| Guld | #FFD700 | 255, 215, 0 | Forlygte |

## STIL
- Flat design / vector
- Ingen skygger
- Ingen gradienter
- Geometriske former
- Simpelt og minimalistisk

## PROPORTIONER
- Kvadratisk canvas (1:1)
- Ikon fylder 64x64 units
- Baggrundscirkel radius: 30 units (93.75% af halv bredde)
- Centreret komposition

## ELEMENTER I IKONET
1. Cirkulær orange baggrund
2. Tuktuk karosseri (hvid)
3. Tuktuk tag (grøn)
4. To hjul (navy blå med hvid midte)
5. Vindue (himmelblå rektangel)
6. Forlygte (lille guld cirkel)

## TEKST (valgfri)
- "Indien" - fed skrift, saffron orange
- "Rejsedagbog" - let skrift, navy blå
- Placeret til højre for ikonet
- Vertikal stabling

## COPY-PASTE PROMPT

```
Circular emblem logo, flat vector style, Indian auto-rickshaw (tuktuk) icon centered on saffron orange (#FF9933) circular background, tuktuk has white body, green (#138808) roof, navy blue (#000080) wheels with white centers, light blue window, small gold headlight, geometric simplified shapes, no shadows, no gradients, clean minimalist design, isolated on white background, professional logo design
```

